The waltz-data-flow-table displays a summary of logical data flows between systems.
Passing in an optional self-id attribute will disable any links to apps with that id.
HTML 
<waltz-data-flow-table
            flows="flows"
            self-id="1">
</waltz-data-flow-table>
JS 
$scope.flows = [
    {
        sourceEntity: { name: 'producer', id: 1},
        targetEntity: { name: 'consumer', id: 2},
        dataTypes: ['PARTY', 'TRADE']
    }, ...
];
