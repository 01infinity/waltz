# Khartec - Waltz

## Technology Stack

### Server

- Java 8
- Embedded Jetty or WAR file (tested on Tomcat 7/8)
- Spark framework
- JDBC via JOOQ


### Supported Databases

- Postgresql 
- MariaDB


### Client

- Browser based
    - IE 9+, Chrome, Safari, Firefox)
- AngularJS 1.4
- Bootstrap 3
- D3
